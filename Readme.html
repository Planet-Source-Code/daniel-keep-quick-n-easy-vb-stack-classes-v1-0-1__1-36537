Ok, I suppose I should actually write how to use these things, so yeah. ;)

Let's start with a basic example of using each stack. This is actually really, really easy :o

First up, dump the three stack classes CStringStack, CLongStack and CStack into a project. Now, find somewhere in your code to add some dummy code (just for demonstration purposes, like maybe on a command button or something). Put the following in:

Dim sstkTemp As CStringStack
Dim lstkTemp As CLongStack
Dim stkTemp As CStack

Set sstkTemp = New CStringStack
Set lstkTemp = New CLongStack
Set stkTemp = New CStack

 Now that's how you create your stacks. To use them, since they all use the same methods, I'll juse use the long stack.

With lstkTemp
  .Push h1543&
  .Push hA632&
  .Push h24FF&
  ' We just 'pushed' three numbers onto the stack. Now, let's pop them off,
  ' debugging the output as we go
  Debug.Print .Pop
  Debug.Print .Pop
  Debug.Print .Pop
End With

 The above code would generate the following in the Immediate window:

5443
42546
9471

 The only other thing you need to know about is the CStringStack.DumpStack() method. It takes a delimiter as an argument, and dumps the entire stack from bottom to top. For example, a stack with the following pushed on (in that order): "C:", "WINNT", "SYSTEM" that had the following call made: DumpStack("\") would return:

C:\WINNT\SYSTEM
